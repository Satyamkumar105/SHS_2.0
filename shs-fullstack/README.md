## How to use

1. Backend:
   - cd server
   - npm install
   - configure .env
   - npm run seed
   - npm run dev

2. Frontend:
   - cd client
   - use your preferred React setup (Vite recommended)
   - npm install
   - npm run dev

Files included: backend scaffold (server/), minimal frontend scaffold (client/), and original markdown guides.
